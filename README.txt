═══════ 快速上手（30 秒） ═══════

1. 解压这个文件夹到任意位置（例如 D:/tools/mpv.net）
2. 双击 mpvnet.exe，或者把视频文件拖到 mpvnet.exe 图标上即可
3. 想切片按 h/k/e，想连播打开文件夹里任一个视频即可


═══════ 播放器特点 ═══════

▸ 内核强 —— 基于 mpv 0.41，格式支持业内最全，MKV/H.265/HDR/10bit
  /VP9/AV1/FLAC/DTS/杜比全景声 全都能开，不挑素材

▸ 画质高 —— 默认启用 gpu-hq 高画质渲染 + NVIDIA nvdec 硬件解码
  （放老片会做锐化和轻度反交错，比 PotPlayer 默认好看一档）

▸ UI 现代 —— 使用 uosc 皮肤，鼠标移到屏幕底部会自动显示控制栏
  （播放/暂停按钮、播放列表、字幕、音轨、倍速、全屏一应俱全）

▸ 无损切片 —— 内置改造版 trim.lua，配合同目录的 ffmpeg.exe，
  按 h/k/e 三个键即可导出视频/音频片段，不重编码、不损失画质

▸ 便携 —— 所有配置和插件都在 portable_config/ 里，不写注册表、
  不污染系统，随手拷贝到另一台机器就能用

▸ 中文字幕 —— 默认字体 微软雅黑 45pt 金黄，深色描边，看电影/教程都清晰


═══════ 常用操作 ═══════

【播放控制】
- 空格          播放 / 暂停（也可以点屏幕底部控制栏的 ▶/⏸ 按钮）
- 双击          全屏 / 退出全屏
- 右键          呼出菜单（打开文件、切字幕、切音轨等等）
- f             切换全屏
- q 或 Esc      退出

【跳转】
- 方向键 ← →   小步跳（视频按关键帧、音频按秒）
- 方向键 ↑ ↓   大步跳
- Shift+方向    更精细 / 更大步长
- 点进度条      拖到任意位置
- Home / End    跳到最前 / 最后

【音量】
- 鼠标滚轮      调整音量
- m             静音

【字幕/音轨】
- 屏幕底部控制栏点 "字幕" / "音轨" 图标切换
- 或按 j 切字幕、# 切音轨

【截图】
- s             普通截图（PNG，保存到当前目录）
- S             不含字幕截图


═══════ 播放列表：文件夹连播 ═══════

★ 已开启「自动加载同目录文件」

打开一个视频后，同文件夹里其他视频/音频**自动加进播放列表**，
不用手动拖。播完当前会自动播下一个。

【连播控制】
- >              下一集
- <              上一集
- 屏幕底部 ⬅ ➡   上一个 / 下一个
- 屏幕底部 🔁    切换循环模式（不循环 / 单曲 / 列表）

【打开播放列表（左上角悬浮）】
- TAB            打开 / 关闭播放列表
- ↑ ↓            移动光标
- Enter          播放光标所在项
- Backspace      从列表移除
- Esc            关闭
（方向键仅在列表打开时用于导航，关闭后恢复正常跳转）

【手动加视频】
- 拖文件到 mpvnet 窗口 → 追加到列表末尾


═══════ 无损视频/音频切片（本版核心功能） ═══════

【原理】
调用同目录 ffmpeg.exe 用 -c copy 参数切片，画质无损、速度极快
（1 分钟视频切片通常在 1 秒内完成）

【操作】
1. 打开视频/音频文件
2. 播放到想要的起点 → 按 h → OSD 提示已设起点
3. 播放到想要的终点 → 按 k → OSD 提示已设终点
4. 按 e → 导出

导出的切片写到源文件**同目录**，自动编号：
    原文件名 1.mp4
    原文件名 2.mp4

【切片辅助键】
- Shift+H       跳回已设起点复听 / 复看
- Shift+K       跳回已设终点复听 / 复看
- t             切换是否剥离元数据（默认不剥离，保留原信息）

【关于起点会"跳到前一帧"】
视频无损切片的物理限制：起点必须落在关键帧（I 帧）上。
如果非要精确到某一帧，只能重编码，那就不无损了。音频不受影响。

═══════ 常见问题 ═══════

Q1: 屏幕底部没看到控制栏？
A:  鼠标移到窗口下 1/4 区域会自动弹出。不动 3 秒会自动隐藏。

Q2: 打开视频黑屏 / 花屏？
A:  个别显卡驱动和 nvdec 不兼容。编辑 portable_config/mpv.conf，
    把 `hwdec=nvdec` 改成 `hwdec=no` 保存重启。

Q3: 想让播放器记住上次看到哪 / 下次继续播？
A:  编辑 portable_config/mpv.conf，把 `save-position-on-quit=no`
    改成 `save-position-on-quit=yes`。

Q4: 按 e 切片没反应 / 报 "Subprocess failed"？
A:  检查 mpvnet.exe 同目录下是不是有 ffmpeg.exe。脚本会依次找：
    ① mpvnet.exe 同目录 → ② portable_config/ → ③ 系统 PATH
    都找不到才报错。

Q5: 想切换成 mpv 官方界面 / 关掉 uosc？
A:  改 portable_config/mpv.conf 里 `osc=no` 为 `osc=yes`，
    然后在 portable_config/scripts/ 里把 uosc/ 文件夹重命名或删除。

Q6: 想改字幕字体 / 大小 / 颜色？
A:  编辑 portable_config/mpv.conf 里的 sub-* 项，重启生效。

═══════ 定制说明（本版和官方 mpv.net 的区别） ═══════

1. 内置 ffmpeg.exe（gyan.dev essentials 8.0.1），切片无需外部依赖
2. trim.lua 已修改：
   • 键位改成 h/k/e（原版是同一键按两次导出，不直观）
   • ffmpeg 路径智能查找（同目录 → portable_config → PATH）
   • 修掉 Windows 上误触发的 macOS 通知红字
   • 修正子进程失败判定，出错时给准确原因
3. uosc 控制栏加了播放/暂停按钮（原版没有）
4. 开启了 autoload：打开视频即自动加载同目录其他视频进列表
5. 精简了体积：删了 macOS/Linux 二进制、调试符号、多语言冗余包
