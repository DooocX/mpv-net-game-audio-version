# player-launcher 使用说明

极简播放列表启动器，配合 mpv.net 使用。双击视频文件即可弹出，通过命名管道 IPC 让 mpv 无缝换片。

---

## 快速上手

1. 把 `player-launcher.exe` 放到 **mpv.net 根目录**（和 `mpvnet.exe` 同级）
2. 右键任意视频 → 打开方式 → 浏览到 `player-launcher.exe` → 勾"始终使用此应用"
3. 之后双击任意视频文件即弹出列表窗口，同目录其他视频自动加载

不做文件关联也 OK：直接双击 `player-launcher.exe` 打开，然后拖视频进窗口。

---

## 快捷键

| 键 | 作用 |
|---|---|
| ↑ ↓ | 移动光标 |
| Enter / 双击 | 播放当前行；播放后光标停在此行 |
| Delete | 把当前文件移到回收站（弹确认框） |
| Ctrl + L | 手动把 launcher 贴到 mpv 窗口左侧（全局生效） |
| 拖入文件 | 加载该文件所在目录 |
| 拖入文件夹 | 加载该文件夹 |
| 点击表头 | 按该列排序，再点切换升/降序 |

---

## 界面

- **搜索框**：输入关键字实时过滤（子串匹配、不区分大小写）
- **打开文件夹** / **刷新**：切换或重扫目录
- **▶ 前缀**：正在播放的行
- **灰色文字**：已经播完过（自然播完到片尾才算，中途切换不算）
- **窗口置顶**：launcher 常驻最上层，mpv 全屏也能看见

---

## 功能

| 功能 | 说明 |
|---|---|
| 无缝切换 | 通过 IPC 让 mpv 常驻，切换视频不重启进程 |
| 自动连播 | 视频自然播完 → 播下一项；最后一项 → 自动暂停 |
| 自动 Snap | 每次 loadfile 时 launcher 自动贴到 mpv 左侧 |
| 时长显示 | MediaInfo.dll 异步扫描，大目录不卡 |
| 文件大小 | 自动 B/KB/MB/GB 单位 |
| 已看灰化 | 记忆已看视频，重启保留（存 `watched.json`） |
| 表头排序 | 4 列都支持点击，不打乱 ▶ 标记 |
| 窗口自适应 | 拖动边框调整，控件跟着延展 |

## 常见问题

**Q: 双击视频没反应？**
A: 检查 `player-launcher.exe` 是否与 `mpvnet.exe` 同目录。

**Q: 打开后 mpv 起来了但没播视频？**
A: 检查 `portable_config/mpv.conf` 里有没有：
```
input-ipc-server=\\.\pipe\mpvnet_launcher
```

**Q: 灰色标记怎么清空？**
A: 关 launcher → 删掉同目录的 `watched.json` → 重开。

**Q: Delete 键删错了？**
A: 文件在**回收站**里，右键还原。永远不会直接删。

**Q: 想让 launcher 不置顶？**
A: 目前 launcher 常驻置顶，不能关闭（这是设计取舍，为了 mpv 全屏时也能看到）。

**Q: 分发给同事？**
A: 打包整个 mpv.net 目录（含 `player-launcher.exe`），解压即用。
